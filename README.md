# Nexo Local — Showcase Pro SPA

Vitrina interactiva de plantillas web para PYMES en Reforma, Chiapas.

## Despliegue en GitHub Pages

1. Crea un repositorio en GitHub (ej. `nexolocal-showcase`)
2. Sube todos los archivos de esta carpeta
3. Ve a **Settings → Pages → Source → Deploy from branch → `main` → `/root`**
4. Tu sitio estará disponible en `https://<tu-usuario>.github.io/nexolocal-showcase/`

## Dominio personalizado (opcional)

Agrega un archivo `CNAME` con tu dominio, ej:
```
showcase.nexolocal.lat
```

## Estructura
```
index.html   ← SPA principal (todo en un archivo)
.nojekyll    ← Evita que GitHub use Jekyll
CNAME        ← (opcional) dominio personalizado
```
